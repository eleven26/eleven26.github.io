此命令中使用的选项具有以下含义：

  • -t - 显示 TCP 端口。
  • -u - 显示 UDP 端口。
  • -n - 显示数字地址而不是解析主机。
  • -l - 只显示监听端口。
  • -p - 显示监听进程的PID和名称。仅当您以 root 或sudo 用户身份运行命令时才会显示此信息。
