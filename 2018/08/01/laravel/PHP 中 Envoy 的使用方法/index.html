envoy 是什么？

envoy 是一个支持 blade 语法的 ssh 远程命令执行的工具。具体来说就是，通过配置 ssh 的账号、密码、key 这些，然后可以使用 envoy 运行一些预定义的命令（比如 git 更新什么的）。
